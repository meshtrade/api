# Feedback & Support

Thank you for using the Mesh API! We appreciate your feedback and want to help you succeed with your integration.

## Found an Issue?

If you encounter bugs, documentation errors, or have suggestions for improvement, please report them on GitHub:

**[Report Issues on GitHub](https://github.com/meshtrade/api/issues)**

When reporting an issue, please include:
- Clear description of the problem
- Steps to reproduce (if applicable)
- Expected vs. actual behavior
- SDK version and language you're using

## Want to Contribute Code?

For developers interested in contributing to the Mesh API codebase, please see our technical contribution guide:

**[Technical Contributing Guide](https://github.com/meshtrade/api/blob/master/CONTRIBUTING.md)**

This guide covers:
- Development environment setup
- Code generation workflows
- Testing procedures
- Pull request process

## Need Help?

- **Documentation**: Start with our [API Reference](/docs/api-reference/) and [Architecture guides](/docs/architecture/service-structure)
- **Examples**: Check the code examples in each API method's documentation
- **GitHub Discussions**: Join the community discussion for questions and best practices

We're committed to providing excellent developer experience and appreciate your input!
