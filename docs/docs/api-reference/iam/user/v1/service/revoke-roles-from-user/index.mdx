## Method Details

**Description:** Revoke roles from an existing user within the authenticated group context.

The role revocation removes the permissions associated with that role from
the user within the group hierarchy. The user will no longer be able
to perform operations that require the revoked role.

**Required Roles:** Check proto file for roles

**Parameters:**
- `Name` (string) (required): Name of the user to revoke roles from in the format users/\{ULIDv2\}.
- `Roles` (string): Roles to revoke from the user in the format groups/\{ULIDv2\}/roles/\{role_id\}.
The role_id corresponds to a value from the meshtrade.iam.role.v1.Role enum.

**Returns:** User

**Method Type:** Unknown
